# Script
## Vowels
 - a /-
 - e -\
 - o -/
 - u \-

Can be mixed:
 - (ae) /-\ - ä
 - (oe) -> - ö
 - (ue) \-\ - ü
 - (ao) /-/ -
 - (ou) <- - w

Add ' to aspirate them
## Konsonants
### Plosive
 - p/b /\
 - t/d //
 - k/g <
### Nasal
 - m >
 - n \\
 - ng \/
### Fricative
 - f/v \>
 - sh/j </
 - s/z <\
 - ch/r />
### Other
 - h/l O

The voicing can be changed from unvoiced x/- to voiced -/x, with exception of the nasal consonants.
